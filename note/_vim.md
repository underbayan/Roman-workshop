#### basic Tips

| Commands | Description |
| ------ | ----------- |
| G   | 跳到最后一行 |
| $   | 跳到该行的最后一个字符 |
| dd   | 删除该行 |
| yy   | 复制该行 |
| p   | 粘贴到下面一行 |
| 6,9 co 12   | 6-9复制到12行下面 |
| 3,100d   | 删除3-100行 |
| /search_name   | 寻找到search_name |
| :set number   | 显示行数 |
